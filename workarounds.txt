# Workaround for using host device/emulator on MacOS from container with Flutter (because docker cant't mount usb on MacOS)
#   On host:
#   1. connect device or start emulator
#   2. get adb devices
#     adb devices  
#   3. forward host port 30000 to device port 50000 for connecting to VM Service (Dart VM) (ports can be any but we'll use ones below)
#     adb -s <device-id> forward tcp:30000 tcp:50000
#
#   In container: 
#   1. forward adb port to host adb server
#     ssh -T -N -f -4 -L5037:localhost:5037 user@host.docker.internal
#
#   2. forward vm service port to host
#     ssh -T -N -f -4 -L30000:localhost:30000 user@host.docker.internal
#
#   3. run flutter app wih --host-vmservice-port
#     flutter run --host-vmservice-port=30000
